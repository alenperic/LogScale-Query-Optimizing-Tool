# LogScale Query Optimizing Tool

An interactive, web-based tool for building and optimizing LogScale queries with real-time feedback, best practice recommendations, and a user-friendly interface.

## Overview

The **LogScale Query Optimizing Tool** is designed to help users construct efficient and optimized LogScale queries with ease. By providing an intuitive interface, real-time optimization, and helpful suggestions, this tool streamlines the process of query building and ensures that your queries follow best practices for performance and efficiency.

## Features

- **Interactive Query Builder**
  - Insert common filters, operators, limiters, and regex functions using intuitive buttons.
  - Utilize the built-in cheat sheet for quick access to syntax and examples.
- **Real-Time Optimization**
  - View the optimized version of your query side-by-side as you build it.
  - Receive instant feedback on potential performance issues.
- **Warnings and Recommendations**
  - Get alerts for resource-intensive operations or suboptimal query patterns.
  - Apply recommended fixes automatically with a single click.
- **Undo and Clear Functions**
  - Use the "Revert" button to undo your last actions (up to 10 previous states).
  - Clear your query input instantly with the "Clear" button.
- **Cheat Sheet and Examples**
  - Access a comprehensive cheat sheet with syntax guidelines and helpful tips.
  - Insert example queries directly into the input for quick modifications.
- **Responsive Design**
  - Enjoy a seamless experience across different devices and screen sizes.

## Getting Started

### Prerequisites

- A modern web browser (Chrome, Firefox, Edge, etc.)

### Installation

1. **Clone the repository**

   ```bash
   git clone https://github.com/alenperic/LogScale-Query-Optimizing-Tool.git
   ```

2. **Navigate to the project directory**

   ```bash
   cd LogScale-Query-Optimizing-Tool
   ```

3. **Open the tool in your web browser**

   - Open the `index.html` file in your preferred web browser.

### Usage

#### Building a Query

- **Use the Sidebar Buttons**

  - Insert filters, operators, limiters, and regex functions by clicking the buttons on the left sidebar.

- **Type or Paste Your Query**

  - You can also type or paste your query directly into the input box on the left.

#### Optimizing the Query

- **Real-Time Optimization**

  - As you build or modify your query, the optimized version will appear in the output box on the right.

- **Handling Warnings**

  - If the tool detects potential issues, a warning will appear at the bottom of the screen.
  - Click "Apply Recommendations" to automatically adjust your query based on best practices.

#### Undo and Clear

- **Revert Changes**

  - Use the "Revert" button to undo your last actions (supports up to 10 previous states).

- **Clear Input**

  - Click the "Clear" button to reset the input box and start fresh.

#### Cheat Sheet and Examples

- **Access Syntax Help**

  - Scroll down to the cheat sheet section for syntax guidelines and helpful tips.

- **Insert Example Queries**

  - Click on any example query in the cheat sheet to insert it directly into the input box.

#### Copying the Optimized Query

- **Copy to Clipboard**

  - Click the "Copy to Clipboard" button under the optimized query to copy it for use in your LogScale environment.
